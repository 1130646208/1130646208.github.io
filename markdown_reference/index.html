    Markdown 漂亮格式¶

    Lists¶

    Unordered lists¶

    Example

    - level 1
    * level 2
        + level 3

- level 1
    - level 2
        - level 3

    Result

    • level 1

      • level 2
        • level 3

    • level 1

      • level 2
        • level 3

    Ordered lists¶

    Example

    1. level 1

    1. level 2.1

    2. level 2.2
        1. level 2.2.1

    Result

    1. level 1

      1. level 2.1

      2. level 2.2

        1. level 2.2.1

    Tasklists¶

    Example

    - [x] checked
- [ ] unchecked
    * [x] checked
    * [ ] unchecked
- [ ] unchecked

    Result

    • checked
    • unchecked
      • checked
      • unchecked
    • unchecked

    Adding line numbers¶

    Example

    ```python linenums="1"
def bubble_sort(items):
    for i in range(len(items)):
        for j in range(len(items) - 1 - i):
            if items[j] > items[j + 1]:
                items[j], items[j + 1] = items[j + 1], items[j]
```
